MASM
----
- Apply 6.11d patch by copying PATCHES\ML611D.EXE to D:\MASM611\BIN and executing it, the file can be deleted after the patch is applied.

DOSBox Conf (macOS location: ~/Library/Preferences/DOSBox 0.74-3-3 Preferences)
-----------
[dosbox]
machine=cga
[render]
scaler=normal3x
[cpu]
cputype=386_slow
[autoexec]
@echo off
# windows
MOUNT C C:\j\msdos
MOUNT D C:\Users\USERNAME\project\ibmpc
# mac
MOUNT C ~/j/msdos
MOUNT D ~/project/ibmpc
SET PATH=Z:\;D:\MASM611\BIN;D:\debugx
SET TMP=D:\TMP
d:
cd tetris

NMake
-----
- Build project with:   b.bat
- Clean project with:   c.bat
- Run binary with:      r.bat

Git BASH
--------
- Add user name and email so you can submit to repository:
    git config --global user.name "USERNAME"
    git config --global user.email "EMAIL"

MartyPC
-------
- Install Rust (https://www.rust-lang.org/tools/install)
- Build release version with "cargo build --release" from the command line in the main folder. This will output the binary to ".\target\release\martypc.exe", copy it to where you want the emulator to be, the other files in this folder are not important.
- Build debug version with "cargo build".
- Clean debug and release builds with "cargo clean".
- Copy the contents of the "install" folder to the root folder of the emulator.
- Download ROMs from (https://github.com/86Box/roms), copy the contents of the "ibmpc82" and "ibmxt86" folder to the "rom" folder of the emulator.
- Download MS-DOS 3.30 disk 1 and 2 images from (https://www.pcjs.org/software/pcx86/sys/dos/microsoft/3.30/) and copy them to the "floppy" folder of the emulator.
